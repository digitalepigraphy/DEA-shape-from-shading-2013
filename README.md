# DEA-shape-from-shading-2013

This is the final source code of the Digital Epigraphy Toolbox project developed as part of the NEH grant HD-51214-11: https://apps.neh.gov/publicquery/AwardDetail.aspx?gn=HD-51214-11

More details about this project can be found at: https://digitalepigraphy.github.io/legacy/

And at the official report of the project published in Humanities Commons, February 28, 2013, pp. 1-11. https://dx.doi.org/10.17613/M64W9R

The results of 3D digitization projects that used the Digital Epigraphy Toolbox can be found at: https://digitalepigraphy.github.io/legacy/toolbox/
